## 题目
```
# 在一次RSA密钥对生成中，假设p=473398607161，q=4511491，e=17
# 求解出d
```
* 本质还是求解e关于phi(pq)的逆元
* 这里可以安装gmpy2的库,也可以用一些在线的工具,如果以后还要使用建议安装这个库
* python脚本
```
import gmpy2
p = 473398607161
q = 4511491
e = 17
d = gmpy2.invert(e, (p-1)*(q-1))
print(d)
# d =125631357777427553
```
最后```d = 125631357777427553```即为我们所求的答案
